Run quality

Assessment checks

  • Preservation fractions between 0.7 and 0.9 usually make the sensitivity plot informative.
  • Synthesis is strongest when historical studies measure comparable QoS behaviour.
  • Block bootstrap is useful when observations are temporally dependent.

Assumptions and limitations

Per-metric NI decisions are computed before aggregation. Preservation fraction controls strictness. Weighted mode depends on stakeholder weights and threshold choices. Block bootstrap is a dependence-aware robustness check, not full time-series modeling.
